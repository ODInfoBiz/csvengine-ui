The CSV Engine is a collection of tools and services for processing and enriching CSV files.

Services
  • A service to parse and clean various types of CSV derivations (or character-separated-values files). The cleaned file is UTF-8 encoded and uses the "," as value separator.
  • This is a prototypical form for generating metadata about a submitted CSV file. The metadata is compliant with the CSV on the Web metadata specification.
